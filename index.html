FilterPanel

The filter component is designed to take a dataset as input and display a list of facets with selectable values.

$('.filter').filterpanel();

Options

Name Type Default Description
dataset Array [] An array of objects to be filtered. The objects' properties determine the facet values displayed in the FilterPanel.
facets Array [] An array of objects representing the filterable facets of the dataset.

Events

As the user selects facet values, the widget triggers events that include the filtered dataset and selected facets.

Name Description
filterpanel:selection:changed Triggered whenever a facet is selected or deselected in the panel. The event handler is passed an event object and a data object containing data and facets properties that represent the current selection state of the filter. 
$('.filter').filterpanel(options)
.on('filterpanel:selection:changed', function(event, data){
    //... do something cool
});

The Tip

A configurable tooltip component (based on Bootstrap's popover)

Required

Name Type Default Description
content string null The HTML content to be shown in the tooltip.

Options

Name type default description
hoverInContent boolean false If true, allows the mouse to hover inside the tooltip content without closing the tooltip.
maxHeight number null If this is defined, the tooltip will have a max height set. If the content is higher than the tooltip container, a tinyscrollbar will be automatically created.
thumbSize number 50 Thumb Size of the scrollbar.
classes string '' Space deliminated classes to be added to the tooltip's tip().
triggerEvent string 'hover' Either 'click' or 'hover'.

Usages

Regular tooltip 
$('#reg-tip').hover_tooltip({
    'content': 'Hello world',
    'container': '.tooltips'
});
Hover in content tooltip 
$('#hover-tip').hover_tooltip({
    'content': 'Hover in content',
    'container': '.tooltips',
    'hoverInContent': true
});
Scrollbar added when maxHeight is set 
$('#scrollbar-tip').hover_tooltip({
    'content': 'Lorem ipsum ...',
    'container': '.tooltips',
    'hoverInContent': true,
    'maxHeight': 125
});
clickable tooltip 
$('#clickable-tip').hover_tooltip({
    'content': 'Hello world',
    'container': '.tooltips',
    'triggerEvent': 'click'
});

More/Less

Limits the number of child elements shown by default and provides a button to toggle between the limited subset and the entire list.

Options

Name Type Default Description
openByDefault boolean false If true, shows the entire list by default (the "more" state)
minItems Number 2 The number of items to display in the "less" state
minHeight Number If set, this overrides minItems and sets the element to a minimum height in the "less" state rather than showing/hiding child elements. This is useful for limiting the lines of text shown by default.
itemClass String If specified, this is a selector used to filter the child elements to show more/less of. If omitted, all child elements are included.
linkContainer jQuery,
String,
HTMLDOMElement		 The container element for the more/less links; this will be the element itself if left unspecified.
linkClass String A custom CSS class to add to the more/less links.
truncateText Boolean false If true, and a value is specified for minHeight, adds ellipses to blocks of text that are truncated in the "less" state.

Example

Villians
  • Hans Gruber
  • Darth Vader
  • Hannibal Lector
  • Ivan Drago
<ul>
    <li>Hans Gruber</li>
    <li>Darth Vader</li>
    <li>Hannibal Lector</li>
    <li>Ivan Drago</li>
</ul>
<div id="moreless-container"></div>

$('ul').moreless({
    more: 'more villians',
    less: 'less villians',
    linkContainer: '#more-less-container'
});
