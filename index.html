Filter

The filter component is designed to take a dataset as input and display a list of facets with selectable values.

$('.filter').filter();

As the user selects facet values, the widget triggers events that include the filtered dataset and selected facets.

The Tip

A configurable tooltip component (based on Bootstrap's popover)

Required

Name type default description
content string null The HTML content to be shown in the tooltip.

Options

Name type default description
hoverInContent boolean false If true, allows the mouse to hover inside the tooltip content without closing the tooltip.
maxHeight number null If this is defined, the tooltip will have a max height set. If the content is higher than the tooltip container, a tinyscrollbar will be automatically created.
thumbSize number 50 Thumb Size of the scrollbar.
classes string '' Space deliminated classes to be added to the tooltip's tip().
triggerEvent string 'hover' Either 'click' or 'hover'.

Usages

Regular tooltip 
$('#reg-tip').hover_tooltip({
    'content': 'Hello world',
    'container': '.tooltips'
});
Hover in content tooltip 
$('#hover-tip').hover_tooltip({
    'content': 'Hover in content',
    'container': '.tooltips',
    'hoverInContent': true
});
Scrollbar added when maxHeight is set 
$('#scrollbar-tip').hover_tooltip({
    'content': 'Lorem ipsum ...',
    'container': '.tooltips',
    'hoverInContent': true,
    'maxHeight': 125
});
clickable tooltip 
$('#clickable-tip').hover_tooltip({
    'content': 'Hello world',
    'container': '.tooltips',
    'triggerEvent': 'click'
});
